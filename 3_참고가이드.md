# 🐍 03. 파이썬 실습 전 필수 가이드: 규칙과 꿀팁

파이썬은 '보기 좋은 코드가 쓰기에도 좋다'는 철학을 가지고 있습니다. 실습 생산성을 높여주는 필수 정보들을 확인해 보세요.

---

## 1. 파이썬의 생명: 들여쓰기 (Indentation)

파이썬은 자바처럼 중괄호({ })를 쓰지 않습니다. 대신 **공백(Space)**으로 코드의 범위를 구분합니다.

- **규칙**: 같은 범위(블록)에 있는 코드들은 반드시 **수직 정렬**이 맞아야 합니다.
- **표준**: 파이썬 공식 가이드(PEP 8)에서는 **스페이스 4칸**을 권장합니다.
- **팁**: VS Code에서는 `Tab`키를 누르면 자동으로 스페이스 4칸으로 변환해 줍니다.



(들여쓰기 예시)
if True:
    print("이 줄은 들여쓰기가 되어 있어 실행됩니다.")
    print("이 줄도 같은 블록이라 함께 실행됩니다.")
print("이 줄은 들여쓰기가 없어 if문과 상관없이 실행됩니다.")

---

## 2. VS Code 파이썬 필수 단축키

마우스를 덜 쓸수록 코딩 실력은 빨리 늘어납니다. 이것만은 꼭 외워두세요!

| 기능 | Windows 단축키 | 설명 |
| :--- | :--- | :--- |
| **코드 실행** | `F5` 또는 `Ctrl + F5` | 작성한 파이썬 파일을 즉시 실행합니다. |
| **줄 복사** | `Shift + Alt + ↓ / ↑` | 현재 커서가 있는 줄을 아래나 위로 복사합니다. |
| **줄 이동** | `Alt + ↓ / ↑` | 현재 줄을 위아래로 자유롭게 이동시킵니다. |
| **주석 처리** | `Ctrl + /` | 선택한 영역을 한 번에 주석으로 만들거나 해제합니다. |
| **터미널 열기** | `Ctrl + ` ` (백틱) | 아래쪽 터미널 창을 보였다가 숨겼다가 합니다. |
| **자동 완성** | `Ctrl + Space` | 코드를 입력하다가 추천 목록을 다시 띄웁니다. |

## 3. 코드의 메모장: 주석 (Comments)

나중에 코드를 다시 봤을 때나 동료에게 설명할 때 사용합니다. 파이썬 인터프리터는 주석을 무시하고 실행하지 않습니다.

- **한 줄 주석**: `#` 기호를 사용합니다.
- **여러 줄 주석**: 따옴표 세 개(`'''` 또는 `"""`)로 감쌉니다.

(주석 예시)
`#` 이것은 사람에게만 보이는 설명입니다.
print("Hello World")  `#` 코드 옆에 쓸 수도 있습니다.

---

## 4. 파이썬의 작명 규칙 (Naming Convention)

변수나 함수의 이름을 지을 때 파이썬 개발자들 사이의 약속입니다.

- **Snake Case**: 모든 글자를 소문자로 쓰고 단어 사이를 언더바(`_`)로 연결합니다.
  - 예: `user_name`, `student_score` (파이썬 권장 방식)
- **자료형 주의**: `print`, `int`, `if` 처럼 이미 파이썬에서 사용 중인 예약어는 변수 이름으로 쓸 수 없습니다.

---

## 5. 실습 시 주의할 점 (꿀팁)

1. **파일 저장**: VS Code 왼쪽 파일 이름 옆에 **흰색 점**이 보인다면 아직 저장되지 않은 상태입니다. 반드시 `Ctrl + S`로 저장 후 실행하세요.
2. **확장자 확인**: 파일 이름은 반드시 `.py`로 끝나야 파이썬이 동작합니다.
3. **대소문자 구분**: 파이썬은 `Apple`과 `apple`을 완전히 다른 것으로 인식합니다.
