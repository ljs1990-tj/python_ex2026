# 🛠️ 02. VS Code 파이썬 개발 환경 설정 가이드

파이썬 코드를 작성하고 실행하기 위한 가장 대중적인 도구인 **Visual Studio Code(VS Code)** 설정 방법을 배워봅시다.

---

## 1. 파이썬(Python) 설치하기

VS Code를 설정하기 전, 컴퓨터가 파이썬 언어를 알아들을 수 있도록 '인터프리터'를 먼저 설치해야 합니다.

1. **공식 홈페이지 접속**: python.org 다운로드 페이지 방문.
2. **설치 파일 실행**: 최신 버전의 파이썬 설치 파일을 실행합니다.
3. **중요 체크 포인트**: 설치 창 하단의 **[Add Python to PATH]** 체크박스를 반드시 선택하세요! 
   - (이 설정을 해야 어디서든 파이썬을 실행할 수 있습니다.)

### 파이썬 설치 시 **'Add Python to PATH'** 체크를 깜빡했다면, 터미널에서 `python` 명령어를 인식하지 못합니다. 재설치 없이 윈도우 설정에서 직접 경로를 추가하는 방법은 99_환경변수설정.md 파일을 참고해주세요.



---

## 2. VS Code에서 파이썬 확장팩 설치

VS Code는 단순한 메모장과 같습니다. 파이썬 전용 기능을 추가하기 위해 '확장 프로그램(Extension)'을 설치해야 합니다.

1. **Extensions 아이콘 클릭**: VS Code 왼쪽 사이드바에서 블록 모양 아이콘을 클릭합니다. (단축키: Ctrl + Shift + X)
2. **검색**: 상단 검색창에 `Python`을 입력합니다.
3. **설치**: Microsoft에서 제공하는 **Python** 확장 프로그램을 찾아 [Install] 버튼을 누릅니다.
   - 이 프로그램이 설치되면 코드 자동 완성, 문법 검사 등의 기능을 사용할 수 있습니다.



---

## 3. 첫 번째 파이썬 파일 만들기

환경 설정이 잘 되었는지 확인하기 위해 간단한 파일을 만들어 봅시다.

1. **폴더 열기**: [File] > [Open Folder]를 선택해 코드를 저장할 폴더를 하나 만듭니다.
2. **새 파일 생성**: 왼쪽 탐색기 빈 공간을 우클릭하거나 새 파일 아이콘을 눌러 `hello.py` 파일을 만듭니다.
   - **주의**: 파이썬 파일은 반드시 이름 뒤에 **`.py`** 확장자가 붙어야 합니다.
3. **코드 작성**: 아래 내용을 입력하세요.
   - print("Hello, Python!")

---

## 4. 코드 실행하는 방법

파이썬 코드를 실행하는 방법은 크게 두 가지가 있습니다.

### 방법 ①: 실행 버튼 클릭 (추천)
- 편집기 오른쪽 상단에 있는 **[▷ (Run Python File)]** 버튼을 클릭합니다.
- 하단 터미널 창에 결과가 출력되는지 확인합니다.

### 방법 ②: 터미널 명령어로 실행
- VS Code 하단 터미널 창에 직접 입력합니다.
- 입력값: python hello.py



---

## 5. 자주 발생하는 문제 해결 (Troubleshooting)

### Q. 'python'은(는) 내부 또는 외부 명령이 아닙니다...
- **원인**: 파이썬 설치 시 'Add Python to PATH'를 체크하지 않았을 때 발생합니다.
- **해결**: 파이썬을 삭제 후 다시 설치하거나, 시스템 환경 변수에 파이썬 경로를 직접 추가해야 합니다.

### Q. 인터프리터가 선택되지 않았다는 메시지가 떠요.
- **해결**: VS Code 하단 상태 표시줄의 [Select Interpreter]를 클릭하거나, `Ctrl + Shift + P`를 누른 뒤 `Python: Select Interpreter`를 검색하여 설치된 파이썬 버전을 선택해 주세요.
