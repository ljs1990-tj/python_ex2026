# 🔢 03-1. 파이썬 연산자 (Operators)

연산자는 데이터를 계산하거나 비교하여 새로운 값을 만들어내는 기호입니다. 조건문(`if`)이나 반복문(`while`)의 조건을 만들 때 필수적으로 사용됩니다.

---

## 1. 산술 연산자: "수학 계산하기"

가장 기본적인 사칙연산과 파이썬 특유의 계산 기호들입니다.

| 연산자 | 기능 | 예시 (a=10, b=3) | 결과 |
| :--- | :--- | :--- | :--- |
| `+` | 더하기 | a + b | 13 |
| `-` | 빼기 | a - b | 7 |
| `*` | 곱하기 | a * b | 30 |
| `/` | 나누기 | a / b | 3.333... |
| `//` | **몫** 구하기 | a // b | 3 |
| `%` | **나머지** 구하기 | a % b | 1 |
| `**` | **거듭제곱** | a ** b | 1000 (10의 3승) |



---

## 2. 비교 연산자: "참인가 거짓인가?"

두 값을 비교하여 결과로 불리언(`True` 또는 `False`)을 내놓습니다. 조건문의 핵심입니다.

- `==` : 같다 (예: `a == b`)
- `!=` : 같지 않다 (예: `a != b`)
- `>` , `<` : 크다, 작다
- `>=` , `<=` : 크거나 같다, 작거나 같다

---

## 3. 논리 연산자: "조건 결합하기"

여러 개의 조건을 하나로 묶을 때 사용합니다. 자바의 `&&`, `||`, `!` 대신 영어 단어를 그대로 사용합니다.

- **`and`**: 모든 조건이 참이어야 참 (자바의 `&&`)
- **`or`**: 하나라도 참이면 참 (자바의 `||`)
- **`not`**: 참을 거짓으로, 거짓을 참으로 뒤집음 (자바의 `!`)

(예시)
if score >= 80 and score <= 100:
    print("합격권입니다.")

---

## 4. 복합 대입 연산자: "계산과 저장을 동시에"

변수의 값을 업데이트할 때 코드를 줄여줍니다.

- `a += 1` : `a = a + 1`과 동일
- `a -= 2` : `a = a - 2`과 동일
- `a *= 3` : `a = a * 3`과 동일

---

## 5. 멤버십 연산자: "그 안에 들어있니?" (중요!)

파이썬에서 리스트나 딕셔너리를 다룰 때 가장 편리한 연산자입니다.

- **`in`**: 값이 목록 안에 있으면 참
- **`not in`**: 값이 목록 안에 없으면 참

(예시)
fruits = ["사과", "바나나"]
if "사과" in fruits:
    print("사과가 바구니에 있습니다.")

---

## 💡 초보자를 위한 핵심 팁

1. **`=` vs `==`**: `=`는 값을 넣는 것(대입), `==`는 같은지 물어보는 것(비교)입니다. 절대 헷갈리지 마세요!
2. **나머지 연산자(`%`)의 마법**: 어떤 숫자를 2로 나눈 나머지가 0이면 **짝수**, 1이면 **홀수**입니다. 이 원리는 프로그래밍에서 정말 자주 쓰입니다.
3. **우선순위**: 수학과 마찬가지로 곱셈/나눗셈이 덧셈/뺄셈보다 먼저 계산됩니다. 헷갈릴 때는 괄호`( )`를 적극 사용하세요.
