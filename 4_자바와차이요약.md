# ☕ Java Developer's Guide to 🐍 Python

자바 개발자의 관점에서 본 파이썬 핵심 문법 차이점 정리입니다.

---

## 1. 변수 선언 및 타입 시스템
파이썬은 **동적 타이핑** 언어이며, 모든 데이터는 객체로 취급됩니다.

* **타입 선언 생략:** 변수명에 바로 값을 할당합니다.
* **None:** 자바의 `null`에 해당합니다. (첫 글자 대문자)
* **Boolean:** `True`, `False`로 표기합니다. (첫 글자 대문자)

```python
# Java: int score = 100;
score = 100

# Java: String name = "Alice";
name = "Alice"

# Java: boolean isValid = true;
is_valid = True
```

## 2. 연산자 (Operators)
자바와 유사하지만 증감 연산자가 없다는 점에 주의하세요.

| 구분 | Java | Python |
| :--- | :--- | :--- |
| **논리 연산** | `&&`, `||`, `!` | `and`, `or`, `not` |
| **증감 연산** | `i++`, `++i` | **없음** (`i += 1` 사용) |
| **나누기(실수)** | `5 / 2` (결과: 2) | `5 / 2` (결과: 2.5) |
| **나누기(몫)** | `5 / 2` (결과: 2) | `5 // 2` (결과: 2) |
| **거듭제곱** | `Math.pow(2, 3)` | `2 ** 3` |

## 3. 제어문 (Flow Control)
중괄호 `{}` 대신 **콜론(`:`)**과 **들여쓰기(Indentation)**를 사용합니다.

### If문
```python
# elif 사용에 주의하세요.
if x > 0:
    print("Positive")
elif x < 0:
    print("Negative")
else:
    print("Zero")
```

### For문
자바의 `for-each` 루프와 유사하게 작동합니다.
```python
# 0부터 9까지 반복 (Java: for(int i=0; i<10; i++))
for i in range(10):
    print(i)

# 리스트 순회 (Java: for(String item : items))
items = ["A", "B", "C"]
for item in items:
    print(item)
```

## 4. 데이터 구조 (Collections)
자바의 인터페이스 기반 컬렉션보다 사용법이 훨씬 간단합니다.

| 타입 | Java (Example) | Python (Example) | 특징 |
| :--- | :--- | :--- | :--- |
| **List** | `ArrayList<String>` | `["a", "b"]` | 가변 배열 |
| **Tuple** | (해당 없음) | `("a", "b")` | **변경 불가능(Immutable)** |
| **Dictionary**| `HashMap<K, V>` | `{"key": "value"}` | Key-Value 쌍 |
| **Set** | `HashSet<T>` | `{"a", "b"}` | 중복 불가, 집합 연산 가능 |

## 5. 클래스와 함수 (Class & Method)

### 함수 정의
```python
# Java: public int add(int a, int b) { return a + b; }
def add(a, b):
    return a + b
```

### 클래스 구조
생성자명은 고정되어 있으며, `self`를 통해 인스턴스 멤버에 접근합니다.

```python
class Person:
    # 생성자 (Constructor)
    def __init__(self, name):
        self.name = name  # 인스턴스 변수 (Java의 this.name)

    # 인스턴스 메서드
    def say_hello(self):
        print(f"Hello, I am {self.name}")

# 객체 생성 (new 키워드 없음)
p = Person("JavaUser")
p.say_hello()
```

## 6. 자바 개발자를 위한 핵심 Tip
1.  **문자열 비교:** `==` 연산자가 값 자체를 비교하므로 `.equals()`를 쓸 필요가 없습니다.
2.  **명명 규칙(Naming):** * 변수/함수: `snake_case`
    * 클래스: `PascalCase`
3.  **세미콜론:** 문장 끝에 `;`를 붙이지 않습니다.
4.  **강력한 슬라이싱:** `string[0:3]`이나 `list[::-1]`과 같이 배열을 다루는 강력한 문법을 제공합니다.
