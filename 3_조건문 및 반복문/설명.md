# 🚦 06. 프로그램의 흐름 제어: 조건문과 반복문

프로그램이 상황에 따라 다른 길을 가거나, 똑같은 일을 수만 번 반복하게 만드는 방법을 배워봅시다.

---

## 1. 조건문 (if): "만약 ~라면"

주어진 조건이 참(`True`)인지 거짓(`False`)인지에 따라 실행할 코드를 결정합니다.

- **핵심 포인트**: 조건식 끝에 반드시 **콜론(:)**을 붙이고, 실행할 코드는 **들여쓰기**를 해야 합니다.
- **구조**: `if` (만약) -> `elif` (그게 아니라 이 조건이라면) -> `else` (그 외 나머지 전부)



(조건문 활용 예시)
score = 85

if score >= 90:
    print("A학점입니다.")
elif score >= 80:
    print("B학점입니다.")
else:
    print("재시험 대상입니다.")

---

## 2. 반복문 (for): "정해진 만큼 반복"

리스트나 범위(`range`) 안의 데이터를 하나씩 꺼내며 반복할 때 사용합니다. 파이썬에서 가장 자주 쓰이는 반복문입니다.

- **range(시작, 끝)**: 숫자 범위를 만들어주는 함수입니다. (끝 번호는 포함되지 않음!)
- **특징**: 데이터를 하나씩 뽑아서 변수에 담고, 블록 안의 코드를 실행합니다.



(for문 활용 예시)
# 0부터 4까지 5번 반복
for i in range(5):
    print(i, "번째 반복 중...")

# 리스트 내용 출력
fruits = ["사과", "바나나", "포도"]
for f in fruits:
    print("맛있는", f)

---

## 3. 반복문 (while): "조건이 맞을 때까지 반복"

어떤 조건이 참(`True`)인 동안에는 코드를 계속해서 반복합니다.

- **주의**: 조건이 계속 참이면 무한 루프(Infinite Loop)에 빠질 수 있으므로, 언젠가는 조건이 거짓이 되도록 설계해야 합니다.

(while문 활용 예시)
count = 1
while count <= 3:
    print("카운트:", count)
    count = count + 1  # 이 줄이 없으면 무한 반복됩니다!

---

## 4. 흐름 끊기: break와 continue

반복문 안에서 흐름을 조절하는 특수 명령어입니다.

- **break**: 반복문을 즉시 탈출합니다.
- **continue**: 아래 코드를 무시하고 다음 반복 순서로 건너뜁니다.

(예시)
for i in range(1, 11):
    if i == 5:
        break  # 5가 되면 반복문 종료
    print(i)
