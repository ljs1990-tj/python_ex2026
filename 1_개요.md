# 🐍 01. 파이썬(Python) 개요 및 특징

파이썬은 1991년 귀도 반 로섬(Guido van Rossum)이 발표한 언어로, 오늘날 데이터 분석, AI, 웹 개발 등 거의 모든 분야에서 가장 사랑받는 프로그래밍 언어입니다.

---

## 1. 파이썬의 핵심 철학: "간결함이 복잡함보다 낫다"

파이썬은 사람이 읽기 쉬운 코드를 지향합니다. 자바처럼 중괄호(`{ }`)나 세미콜론(`;`)에 집착하지 않고, **들여쓰기(Indentation)**를 통해 코드의 구역을 나눕니다. 덕분에 코드가 깔끔하고 누가 짜더라도 비슷한 모양을 유지합니다.

---

## 2. 파이썬의 주요 특징

### ① 인터프리터 언어 (Interpreter)
컴파일 과정 없이 코드를 한 줄씩 읽어서 바로 실행합니다. 수정한 내용을 즉시 확인할 수 있어 학습과 개발 속도가 매우 빠릅니다.

### ② 동적 타이핑 (Dynamic Typing)
자바처럼 `int a = 10;`이라고 타입을 미리 정할 필요가 없습니다. 
- 예: `a = 10` (파이썬이 알아서 숫자로 이해함)

### ③ 강력한 라이브러리 (Ecosystem)
"이미 누군가 다 만들어 두었다"는 말이 있을 정도로 방대한 도구 상자(라이브러리)를 가지고 있습니다. 데이터 분석(Pandas), 인공지능(TensorFlow), 웹(Django) 등 원하는 기능을 가져다 쓰기만 하면 됩니다.

---

## 3. 파이썬의 실행 구조

파이썬 코드는 어떻게 실행될까요?

1. **소스 코드 (.py)**: 개발자가 텍스트로 코드를 작성합니다.
2. **파이썬 인터프리터**: 코드를 중간 단계인 '바이트 코드'로 바꿉니다.
3. **PVM (Python Virtual Machine)**: 컴퓨터가 이해할 수 있는 기계어로 바꿔서 실행합니다.



---

## 4. 파이썬 설치 및 환경 설정 체크리스트

파이썬 공부를 시작하기 전, 다음 두 가지가 준비되어야 합니다.

1. **Python 인터프리터**: 파이썬 공식 홈페이지(python.org)에서 설치합니다.
2. **코드 에디터 (IDE)**: 
   - **VS Code**: 가장 대중적이고 가벼운 선택.
   - **PyCharm**: 파이썬 전용 전문 개발 도구.
   - **Jupyter Notebook**: 데이터 분석 학습에 최적화된 도구.
