# 📦 07. 코드의 재사용: 함수 (Function)

함수는 특정 작업을 수행하는 코드들을 하나로 묶어 이름을 붙인 것입니다. 반복되는 노가다를 줄여주고 코드를 깔끔하게 만들어주는 필수 도구입니다.

---

## 1. 함수의 구조: def 키워드

파이썬에서는 '정의하다'라는 뜻의 **`def`** 키워드를 사용하여 함수를 만듭니다.

- **이름**: 함수가 하는 일을 나타내는 이름을 지어줍니다.
- **매개변수(Parameter)**: 함수가 실행될 때 외부에서 전달받는 값입니다. (필요 없으면 비워둘 수 있음)
- **실행문**: 함수가 호출되었을 때 돌아갈 코드입니다. 반드시 **들여쓰기**를 해야 합니다.



(함수 정의 예시)
def welcome(name):
    print("안녕하세요,", name, "님!")
    print("파이썬의 세계에 오신 것을 환영합니다.")

# 함수 호출 (실행)
welcome("철수")

---

## 2. 결과 돌려주기: return

함수가 계산을 마친 후 그 결과를 나에게 다시 돌려줘야 할 때 **`return`**을 사용합니다.

- `return`을 만나면 함수는 즉시 종료되고, 그 뒤의 값(결과값)을 함수를 부른 곳으로 보냅니다.

(return 활용 예시)
def add(a, b):
    return a + b

result = add(10, 20)
print("두 수의 합은:", result)

---

## 3. 기본값 설정 (Default Value)

함수를 호출할 때 값을 전달하지 않아도 기본으로 들어갈 값을 정해둘 수 있습니다.

(기본값 예시)
def introduce(name, age=20):
    print("이름:", name)
    print("나이:", age)

introduce("영희")      # age를 안 써도 기본값인 20이 들어감
introduce("민수", 25)  # 25를 쓰면 기본값이 무시되고 25가 들어감

---

## 4. 함수를 사용하는 이유 (장점)

1. **중복 제거**: 똑같은 코드를 여러 번 복사해서 붙여넣을 필요가 없습니다.
2. **관리 용이**: 코드에 수정사항이 생기면 함수 내부만 고치면 됩니다.
3. **가독성**: 코드의 흐름이 한눈에 들어와 읽기 편해집니다.

2. **매개변수 개수**: 함수를 만들 때 정한 매개변수의 개수와 호출할 때 넣는 값의 개수를 맞춰야 합니다.
