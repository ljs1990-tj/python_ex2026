# 🛠️ 02-1. 파이썬 환경 변수(PATH) 수동 설정 가이드

파이썬 설치 시 **'Add Python to PATH'** 체크를 깜빡했다면, 터미널에서 `python` 명령어를 인식하지 못합니다. 재설치 없이 윈도우 설정에서 직접 경로를 추가하는 방법을 알아봅시다.

---

## 1. 파이썬 설치 경로 찾아내기

먼저 파이썬이 어디에 설치되었는지 정확한 주소를 알아야 합니다.

1. `윈도우 키 + R`을 눌러 실행 창을 켭니다.
2. `appdata`라고 입력하고 확인을 누릅니다.
3. 다음 폴더 순서로 이동합니다: 
   - **Local > Programs > Python > Python312** (숫자는 버전에 따라 다를 수 있음)
4. 폴더 상단 주소창을 클릭하여 **경로를 복사**해 둡니다.
   - 예: C:\Users\YourName\AppData\Local\Programs\Python\Python312

---

## 2. 시스템 환경 변수 편집

이제 복사한 주소를 컴퓨터 시스템에 등록할 차례입니다.

1. 윈도우 검색창에 **'환경 변수'**를 입력하고 **[시스템 환경 변수 편집]**을 클릭합니다.
2. 하단의 **[환경 변수(N)...]** 버튼을 누릅니다.
3. 아래쪽 **[시스템 변수]** 목록에서 **Path**를 찾아 선택한 뒤 **[편집(I)...]**을 누릅니다.



---

## 3. 경로 등록하기 (2개 필수)

가장 중요한 단계입니다. 두 가지 경로를 각각 추가해야 합니다.

1. 오른쪽의 **[새로 만들기(N)]**를 누르고 복사했던 **파이썬 설치 경로**를 붙여넣습니다.
   - 예: ...\Python312
2. 다시 **[새로 만들기(N)]**를 누르고 이번엔 경로 뒤에 **\Scripts**를 붙여서 추가합니다.
   - 예: ...\Python312\Scripts
   - (이걸 추가해야 라이브러리 설치 도구인 `pip`를 사용할 수 있습니다.)



---

## 4. 최종 확인

1. 모든 창에서 **[확인]**을 눌러 설정을 저장합니다.
2. 실행 중이던 **VS Code나 터미널 창을 완전히 닫고 다시 엽니다.** (재시작 필수)
3. 터미널(CMD)에 아래 명령어를 입력하여 확인합니다.
   - **python --version**
   - 버전 숫자가 출력된다면 성공입니다!

---

## 💡 왜 이렇게 하나요?

환경 변수(PATH) 설정은 컴퓨터에게 **"내가 'python'이라고 부르면 이 폴더에 있는 프로그램을 실행해줘"**라고 미리 길을 알려주는 작업입니다. 이 작업이 되어 있어야만 VS Code나 터미널 어디서든 편리하게 파이썬을 다룰 수 있습니다.
